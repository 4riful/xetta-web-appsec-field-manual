# Labs And Training

Use this section when you want practice instead of another link dump.

## Practice Flow

1. Pick one topic from [bug-classes/](../bug-classes/README.md).
2. Read two or three writeups from [awesome-lists/best-bug-bounty-writeups.md](../awesome-lists/best-bug-bounty-writeups.md).
3. Practice the same idea in a lab or CTF environment.
4. Write a short report using [reports/](../reports/README.md).

## Training Buckets

- Practice platforms and labs: use for repetition and muscle memory.
- OSCP-style training: use for methodology, enumeration, and reporting discipline.
- Active Directory: useful for broader pentest context, not the core web path.
- Videos and talks: use when you need explanation before reading source material.
- Mindmaps: use to see relationships between topics.
