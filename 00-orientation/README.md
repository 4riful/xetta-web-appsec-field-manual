# Orientation

## Purpose

This section defines how to use the manual without turning it into random scanning or payload spraying.

## Read First

- [Legal and Scope Discipline](legal-and-scope-discipline.md)
- [How To Use This Manual](how-to-use-this-manual.md)
- [Operator Checklists](operator-checklists.md)

## Operating Principles

- Scope first, tools second.
- Use passive collection before active probing.
- Prefer evidence over volume.
- Treat scanner output as leads, not findings.
- Keep raw source archive intact.
