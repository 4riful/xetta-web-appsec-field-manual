# Report Writing Workflow

## Procedure

1. Confirm the bug is reproducible.
2. Reduce the proof of concept to minimal safe steps.
3. Capture request/response evidence.
4. Explain affected roles, assets, and data.
5. State impact without exaggeration.
6. Provide remediation that maps to root cause.
7. Include retest instructions.
